# React + TypeScript Todo App

A simple fully functional **Todo App** built with **React** and **TypeScript**, styled using **vanilla CSS**.  
It allows users to **add**, **edit**, **view**, and **delete** tasks

---

## Features

- Add new tasks  
- Edit existing tasks  
- View all tasks in a clean list  
- Delete tasks  
- Persistent state management using React hooks  
- Styled with plain CSS

---

## Tech Stack

- **React (Vite)** — UI library  
- **TypeScript** — type-safe code  
- **Vanilla CSS** — lightweight custom styling
