# Simple Todo List (TypeScript)

A basic Todo List web app that lets users add, remove and view tasks dynamically using TypeScript.

---

## Features
- Add new tasks using the input field and **addTodo** function
- Prevents empty tasks from being added, telling the user they can't add empty tasks
- Remove task from the list using the **removeTodo** function
- View current tasks using **displayTodos** function

---

## How to Run
1. Add task by using the **addTodo** function, passing your task to it.
2. Remove task from the list using the **removeTodo** function, passing the task's id.
3. View current tasks using **displayTodos** function
